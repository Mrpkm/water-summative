# Water: The Drivers — Problems — Solutions

A Science Summative website about the world's water — its drivers, problems and
solutions — viewed both globally and locally in England.

## Sections
- **Home** — overview and navigation
- **Interview** — Mr. Andy interview (primary research)
- **Drivers** — Atmospheric, Physical & Geological, Global Systems (El Niño), Anthropogenic
- **Problems & Solutions** — every problem and solution in one place
- **Water Cycle** — overview essay with figures

Open `index.html` in any browser, or visit the published GitHub Pages link.
